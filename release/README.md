# Release Files

## Recommended files

- `twotwo-girl-danmaku-adventure-latest.zip`
- `twotwo-girl-danmaku-adventure-v0.1.1.zip`

## What is inside

These zip files contain the built extension package from `dist/`.

## How to install manually

1. Download `twotwo-girl-danmaku-adventure-latest.zip` or `twotwo-girl-danmaku-adventure-v0.1.1.zip`
2. Unzip it to a local folder
3. Open your Chromium-based browser extension page
4. Turn on Developer Mode
5. Choose "Load unpacked"
6. Select the unzipped folder
