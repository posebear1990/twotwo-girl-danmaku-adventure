# Release Files

## Recommended file

- `twotwo-girl-danmaku-adventure-v0.1.0.zip`

## What is inside

This zip contains the built extension package from `dist/`.

## How to install manually

1. Download `twotwo-girl-danmaku-adventure-v0.1.0.zip`
2. Unzip it to a local folder
3. Open your Chromium-based browser extension page
4. Turn on Developer Mode
5. Choose "Load unpacked"
6. Select the unzipped folder
